# AI Conference Helper - 项目草案

## 一句话定位

一个轻量级 AI 会议分析工具：上传会议录音，获得高质量转录和深度自定义分析，弥补飞书免费版的不足。

---

## 要解决的问题

- 飞书免费版妙记功能有限，AI 摘要质量一般
- 企业版按人收费（¥50-100/人/月），小团队成本高
- 即使付费，飞书也不支持自定义分析维度
- 会议内容锁在平台内，无法自由检索和二次加工

## 不做什么

- 不做视频会议系统（继续用飞书/腾讯会议开会）
- 不做实时转录（MVP 阶段）
- 不做用户体系/权限管理（MVP 阶段）

---

## 核心功能（MVP）

1. **音频上传** — 支持 MP3/M4A/WAV/MP4 等常见格式
2. **语音转文字** — 调用 Whisper API，支持中英文混合
3. **AI 深度分析** — 可自定义 Prompt 模板，包括但不限于：
   - 会议摘要
   - 待办事项（谁 + 做什么 + 截止时间）
   - 关键决策
   - 争议点 / 未解决问题
   - 自定义分析维度
4. **会议历史** — 列表查看、全文搜索
5. **结果导出** — Markdown 格式导出

## 技术方案

```
用户浏览器
    ↓ HTTPS
DMIT Server (1-2 vCPU / 1-2GB RAM)
├── Nginx 反向代理
├── FastAPI 应用（Python）
│   ├── Web 界面（Jinja2 模板）
│   ├── 音频上传 → 转发至 Whisper API
│   ├── 转录文本 → 调用 DeepSeek API 分析
│   └── SQLite 数据库（文本数据）
└── Cloudflare R2（音频文件存储）
```

| 组件 | 选型 | 说明 |
|------|------|------|
| 后端 | FastAPI (Python) | 异步、轻量、自带 API 文档 |
| 前端 | Jinja2 + HTMX | 服务端渲染，无需前端构建工具 |
| ASR | OpenAI Whisper API | 中英文最优，$0.006/分钟 |
| LLM | DeepSeek-V3 API | 中文能力强，价格极低 |
| 数据库 | SQLite | 文本数据极小，单文件够用 |
| 文件存储 | Cloudflare R2 | 免流出费，$0.015/GB/月 |
| 部署 | DMIT 已有服务器 | 与代理服务共存 |

## 工作流

```
飞书/腾讯会议 → 导出录音 → 上传到本工具 → 自动转录 → AI 分析 → 查看/导出结果
```
